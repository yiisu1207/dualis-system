# ERP System

Versión de desarrollo del panel **ERP System** — sistema para gestión de inventario, ventas, finanzas y recursos humanos.

Desarrollado por **Jxsuu** ❤️ — para emprendedores.

Cómo ejecutar localmente:

```bash
npm install
npm run dev
```

Variables de entorno:
- Revisa `src/firebase/config.ts` y ajusta las credenciales de Firebase si las usas.

Despliegue:
- Recomendado: Vercel o Netlify. Conectar el repositorio y configurar la variable de entorno necesaria para Firebase.

Contribuciones:
- Crea un fork y un PR. Ejecuta `npm run dev` para pruebas locales.
